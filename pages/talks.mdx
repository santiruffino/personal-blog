# Git 101

![Git 101 imagen](/static/git-101.png)

Esta charla se realizo el 30/04/2020 en el segundo evento online de [Facebook DevC Cordoba](https://www.facebook.com/groups/DevCCordoba/).

Fue una charla introductoria a lo que es Git, porque usarlo, cual es la diferencia con Github y los comandos basicos para poder crear un repositorio.

Toda la charla fue teorica acompañada de practica donde los asistentes podian ir ejecutando los comandos y participando de forma activa durante la charla.
Se pudo realizar lo siguiente:
- Crear un repositorio
- Agregar, comittear y pushear archivos
- Hacer un fork
- Clonar ese fork
- Editar archivos localmente
- Pushear esos cambios al fork
- Crear un PR
- Resolver conflictos

# Automatizando Google Ads

![Automatizando Google Ads imagen](/static/gdg.png)

Esta charla se realizo el 23/04/2020 en un evento online de [GDG Cordoba](https://www.meetup.com/es/GDG-CORDOBA-AR/).

Fue una charla orientada a gente que trabaje en Marketing y a programadores interesados en el tema.
Ya que basicamente se hablo sobre Google Ads y las formas de automatizar mediante reglas, scripts o API. Se mostraron ejemplos concretos y con casos de negocios para una mayor comprension del publico.
